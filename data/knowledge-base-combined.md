## 1. 앱에 바로 반영할 핵심 진단 로직

### 🔴 과열도(Superheat) 기반 진단 매트릭스

| 과열도 | 의미 | 원인 | 점검 |
|--------|------|------|------|
| **정상 5~12K** (10~20°F) | 증발기 출구에서 냉매가 완전히 기화 후 적정 가열 | 정상 운전 | - |
| **높음 >15K** | 증발기에 냉매 부족, 너무 일찍 기화 완료 | ① 냉매 부족 ② TXV 언더피딩 ③ 필터 막힘 | 사이트글라스 기포, TXV 개도, 필터 ΔT |
| **낮음 <5K** | 액냉매가 컴프레서에 도달 위험 (리퀴드백) | ① TXV 오버피딩 ② 냉매 과충전 ③ 컴프레서 불량(고흡입) | TXV 센싱벌브 위치, 충전량, 컴프레서 전류 |

### 🟠 과냉도(Subcooling) 기반 진단 매트릭스

| 과냉도 | 의미 | 원인 | 점검 |
|--------|------|------|------|
| **정상 4~10K** (8~18°F) | 응축기 출구에서 냉매가 완전히 액화 후 적정 냉각 | 정상 운전 | - |
| **높음 >12K** | 냉매가 응축기에 과다 체류 | ① 배관 막힘(필터/TXV) ② 냉매 과충전 ③ 응축기 과냉 | 필터 전후 ΔT, TXV 작동, 충전량 |
| **낮음 <3K** | 응축기에서 불충분 액화, 플래시 가스 위험 | ① 냉매 부족 ② 응축기 에어플로우 부족 ③ 논콘덴서블(공기 혼입) | 사이트글라스, 팬 작동, 진공 이력 |
| **음수** | 응축기 출구에서 아직 가스 상태 → 매우 심각 | 심각한 저충전 또는 응축기 완전 불량 | 즉시 정지, 누설탐지 |

### 🔵 흡입압력(Suction Pressure) 기반 진단

**정상 범위 (냉매별):**
- R-22: 60~85 psig (4.1~5.9 bar)
- R-410A: 120~145 psig (8.3~10 bar)  
- R-32: 115~145 psig (7.9~10 bar)

**저흡입압력 원인 5가지:**
1. 냉매 부족 → 증발기에 충분한 냉매 없음
2. 에어플로우 부족 → 증발기 열교환 저하 (더티 코일, 팬 불량, 필터 막힘)
3. TXV 막힘/언더피딩 → 냉매 유량 제한
4. 필터 드라이어 막힘 → 배관 제한
5. 증발기 결빙 → 에어플로우 차단 악순환

**핵심 교차 진단:**
- 저흡입 + 저과열 → TXV 불량 또는 플러딩
- 저흡입 + 고과열 → 증발기 스타빙 (냉매 부족 or 제한)

## 2. 냉동 시스템 고장 6대 유형 (앱 진단 플로우 구조)

### Failure 1: 컴프레서 번아웃
**증상:** 기동 불가, 트립, 산성 오일
**원인:**
- 윤활 상실 (냉매 누설로 오일 유실, 리퀴드 플러딩으로 오일 워시아웃)
- 전기적 문제 (전압 불균형, 단락 — 모터 권선 절연 파괴)
- 과열 (더티 응축기, 에어플로우 차단, 팬 불량)
**점검:** 오일 산도 검사, 모터 권선 저항, 전압 불균형

### Failure 2: TXV 고장
**증상:** 과열도 비정상 (너무 높거나 낮음)
**원인:**
- 오리피스 오염물 막힘
- 센싱벌브 충전 가스 누설
- 과열도 조정 과도
**점검:** 과열도/과냉도 측정, 벌브 온도 vs 실제 온도 비교

### Failure 3: 냉매 부족
**증상:** 저흡입압력, 고과열도, 증발기 결빙
**원인:**
- 미세 누설 (접합부, 밸브 스템, 용접 불량)
- 서비스 시 부적절 충전
**점검:** 압력 테스트(질소), 진공, 사이트글라스

### Failure 4: 배관 제한/막힘
**증상:** 고과냉도, 증발기 스타빙
**원인:**
- 필터 드라이어 포화 (수분/산)
- 배관 내 용접 잔재물
- TXV 스트레이너 막힘
**점검:** 필터 전후 온도차 (>3°C면 막힘 의심)

### Failure 5: 논콘덴서블(공기/수분 혼입)
**증상:** 비정상 고압, 냉각능력 저하
**원인:**
- 배관 공사 후 부적절 진공
- 호스 연결 시 공기 유입
**점검:** 고압 vs PT 차트 비교, 냉매 분석

### Failure 6: 전기적 고장
**증상:** 간헐적 트립, 과전류
**원인:**
- 서지/낙뢰
- 접촉기 마모
- 유지보수 불량 (느슨한 터미널)
**점검:** 절연저항, 전류 측정, 터미널 토크

## 3. 앱 시각화 강화 포인트

### 플래시 가스 시각화 (MEP Academy 핵심 콘텐츠)
**위치:** 팽창밸브 직전 (리퀴드 라인)
**정상:** 사이트글라스에 맑은 액체 → 기포 없음
**이상:** 기포/거품 → 플래시 가스 발생
- 원인: 저충전, 과냉도 부족, 배관 열획득

**앱 반영:**
- 응축기→TXV 구간에 "사이트글라스" 인터랙티브 포인트 추가
- 정상: 맑은 파란색 / 이상: 기포 파티클 애니메이션

### 칠러 시스템 확장
**MEP Academy 칠러 콘텐츠 핵심:**
- 공랭식 vs 수냉식 차이 = 열배출 방법 (공기 vs 냉각탑 물)
- 두 개 물 루프: 칠드워터 루프(열 흡수) + 콘덴서워터 루프(열 배출)
- 셸앤튜브 열교환기: 냉매가 튜브 외부, 물이 튜브 내부
- 냉수 공급 44°F(6.6°C), 리턴 54°F(12.2°C) → ΔT 10°F

**앱 반영 (Phase 2):**
- 칠러 전용 사이클 다이어그램
- 물 루프 + 냉매 루프 동시 시각화
- 냉각탑 열배출 프로세스

## 5. 교차 진단 의사결정 트리 (앱 핵심 로직)

```
[증상 관찰]
    │
    ├─ 냉각 불량 ──┬─ 흡입압력 저? ──┬─ 과열도 고? → 냉매 부족 or 배관 제한
    │              │                  └─ 과열도 저? → TXV 불량 (플러딩)
    │              └─ 흡입압력 고? ──┬─ 토출압력 저? → 컴프레서 불량
    │                               └─ 토출압력 고? → 응축기 에어플로우 문제
    │
    ├─ 결빙 ───────── 증발기에 성에 → 에어플로우 점검 → 냉매량 점검 → TXV 점검
    │
    ├─ 소음 ────┬─ 액격음(리퀴드 슬래밍) → 과열도 확인 → TXV/냉매량
    │           └─ 기계적 소음 → 베어링/밸브 플레이트 마모
    │
    ├─ 고압 경보 ──┬─ 응축기 팬 작동? → 팬 모터/릴레이
    │              ├─ 코일 더러운지? → 세척
    │              └─ 논콘덴서블? → PT 차트 비교
    │
    └─ 기동 불가 ──┬─ 전원 확인 → 퓨즈/브레이커
                   ├─ 컨택터 확인 → 코일/접점
                   └─ 컴프레서 → 권선 저항, 절연저항
```

## 6. 앱 UI 개선 제안 (MEP Academy 학습 기반)

### A. "사이트글라스" 인터랙티브 포인트
- 배관의 응축기→TXV 사이에 원형 관찰창 추가
- 정상: 투명 파란색 / 부족: 기포 파티클 / 과충전: 완전 투명

### B. "물의 3상태" 비유 설명 카드
- MEP Academy 스타일: 물→얼음→증기 비유로 냉매 상태 변화 설명
- 각 컴포넌트 터치 시 "여기서 냉매는 ___ 상태" 표시

### C. "열의 이동" 방향 화살표
- 응축기: 냉매→외기 (열 방출 화살표)
- 증발기: 실내공기→냉매 (열 흡수 화살표)
- 현재 프로토타입에는 냉매 흐름만 있고, 열의 이동 방향이 없음

### D. 압력-온도 관계 시각화
- "압력 올리면 온도도 올라간다" → 컴프레서 터치 시 P↑T↑ 애니메이션
- "압력 내리면 온도도 내려간다" → TXV 터치 시 P↓T↓ 애니메이션

## 7. The Engineering Mindset 추가 분석

> 출처: theengineeringmindset.com (Paul Evans, 유튜브 450만+ 구독)
> MEP Academy가 "현장 기사 관점"이라면, Engineering Mindset은 "시스템 엔지니어링 관점" — 칠러/중앙 플랜트 쪽이 특히 강함

### 🏗️ 칠러 시스템 설계 데이터 (앱 Phase 2 DB용)

**공랭식 칠러 (R-134a 기준 설계 데이터)**
| 포인트 | 위치 | 압력(kPa) | 온도(°C) | 상태 |
|--------|------|-----------|---------|------|
| 1 | 증발기 출구 / 컴프레서 입구 | 356 (3.56 bar) | 5.5 | 저압 약간 과열 증기 |
| 2 | 컴프레서 출구 | 915 (9.15 bar) | 43.6 | 고압 과열 증기 |
| 3 | 응축기 출구 | ~900 | ~35 | 고압 과냉 액체 |
| 4 | 팽창밸브 출구 | ~360 | ~3 | 저압 혼합 |

**수냉식 칠러 운전 조건**
- 칠드워터: 공급 6°C (42.8°F) / 리턴 12°C (53.6°F), ΔT = 6K
- 컨덴서워터: 공급 29°C (84.2°F) / 리턴 35°C (95°F), ΔT = 6K
- 컨덴서 유량 > 증발기 유량 (열배출량이 더 크므로)

### 🔧 TXV 작동 원리 상세 (앱 진단 핵심)

**TXV 3대 힘의 균형 (F1 = F2 + F3)**
1. **F1 (개방력)**: 센싱벌브 압력 → 과열도에 비례 → 다이어프램 누름 → 밸브 열림
2. **F2 (폐쇄력)**: 증발기 압력 → 시스템 저압 → 밸브 닫힘 방향
3. **F3 (폐쇄력)**: 스프링 장력 → 과열도 설정값 조정

**균형 상태:** F1 = F2 + F3 → 정상 운전
**과열도 높음:** F1 > F2+F3 → 밸브 더 열림 → 냉매 증가
**과열도 낮음:** F1 < F2+F3 → 밸브 닫힘 → 냉매 감소

**TXV 고장 3가지 모드:**
1. **스타빙(Starving)** — 과열도 고, 토출온도 고, 컴프레서 과열 보호 트립
2. **플러딩(Flooding)** — 과열도 저, 리퀴드백 → 컴프레서 파손 위험
3. **헌팅(Hunting)** — 과열도 반복 진동, 불안정 운전, 소음

**TXV 트러블슈팅 체크리스트 (Danfoss 공식):**
- [ ] 에어플로우 충분한가?
- [ ] 냉매 충전량/과냉도 정상인가?
- [ ] 상류 제한 없는가? (필터 드라이어)
- [ ] 센싱벌브 정위치 장착? (6시 방향 금지)
- [ ] 센싱벌브 단열 되어있는가?
- [ ] 밸브 막힘 징후 있는가?
- [ ] 센싱벌브에 열 가하면 밸브가 반응하는가?

**핵심 감별:**
| 증상 | 에어플로우 부족 | 냉매 부족 |
|------|---------------|----------|
| 과열도 | 약간 높음 | 매우 높음 |
| 과냉도 | 정상~약간높음 | 매우 낮음 |
| 흡입압력 | 약간 낮음 | 매우 낮음 |

### 🏭 칠러 제어/보호 시스템 (앱 에러코드 연동)

**칠러 기동 전 필수 확인 (앱 "칠러 시작 체크리스트" 기능):**
1. 칠드워터 유량 확인 → 미확인 시 기동 불가
2. 컨덴서워터 유량 확인 → 미확인 시 기동 불가
3. 오일 섬프 펌프 작동
4. 모터 권선 온도

**칠러 보호 코드 (공통):**
- 고압 트립 → 응축기 문제 (에어플로우/수량/오염)
- 저압 트립 → 냉매 부족 또는 증발기 유량 부족
- 모터 과전류 → 전기적 문제 또는 기계적 부하 과다
- 오일 압력 저하 → 오일 펌프/필터/누설
- 기동 횟수 제한 → 인러시 전류 보호 (대형 칠러 시간당 3~4회 제한)

**칠러 로그 데이터 활용 (앱 기능 아이디어):**
- 칠러 컨트롤러가 운전 데이터 기록
- 서비스 엔지니어가 이 로그로 진단
- → 우리 앱에서 "칠러 로그 입력 → 자동 분석" 기능 가능

### 🏛️ 5대 측정값 (Five Pillars) — 앱의 핵심 입력 데이터

**매 서비스콜마다 반드시 측정해야 할 5가지:**

| # | 측정값 | 무엇을 알려주는가 | 정상 범위 |
|---|--------|-----------------|----------|
| 1 | **흡입압력** (Suction Pressure) | 증발기 내 끓는점 온도 → 32°F 이하면 결빙 | 냉매별 상이 |
| 2 | **토출압력** (Head Pressure) | 응축기 내 응축 온도 → 외기+15~25°F | 냉매별 상이 |

| 5 | **ΔT** (Delta T) | 코일 전후 온도차 → 열교환 효율 | 16~22°F (냉방) |

### 🔑 핵심 규칙 (TXV 시스템)

**"TXV는 과열도를 일정하게 유지한다"** — 이것이 깨지는 3가지 조건:
1. TXV 자체 고장
2. TXV에 100% 액체가 공급되지 않음 (과냉도 0)
3. TXV에 필요한 압력차가 부족

→ **과냉도 0 + 과열도 비정상 = TXV 문제가 아니라 공급 문제**

### 📋 4대 과열도/과냉도 조합 진단 (Contracting Business / AC Service Tech)

| 조합 | 과열도 | 과냉도 | 진단 | 점검 |
|------|--------|--------|------|------|
| **A** | 높음 | 낮음 | **냉매 부족** | 누설탐지, 충전량 |
| **B** | 낮음 | 높음 | **냉매 과충전** | 회수, 재충전 |
| **C** | 높음 | 높음 | **고압측 제한** (필터/TXV 막힘) | 필터ΔT, TXV, 프로스팅 |
| **D** | 낮음 | 낮음 | **메터링 오버피딩** (TXV 개방고장/피스톤 과대) | TXV 교체, 피스톤 사이즈 |

> ⚠ 조합 C가 가장 위험한 오진 포인트 — 압력만 보면 냉매 부족처럼 보이지만 실제로는 제한

### 🔧 AC Service Tech — 실전 충전 퀵 레퍼런스

**외기온도별 목표 과열도 (Fixed Orifice 시스템):**
- 75~85°F (24~29°C): 과열도 12~15°F (7~8K)
- 85°F+ (29°C+): 과열도 8~12°F (4~7K)

**TXV 시스템은 과냉도로 충전:**
- 목표 과냉도: 제조사 데이터 (없으면 10°F ±3)

**토출라인 온도 (DLT) — 컴프레서 건강 지표:**
- 정상: 150~225°F (65~107°C) — 냉매/모델별 상이
- 과열도 높으면 DLT도 높음 → 압축비 과대 가능성

## 12. Refrigeration Mentor (Trevor Matthews) — 컴프레서 고장 진단

> 출처: refindustry.com, HVAC School 게스트 출연 (Emerson/Copeland 전문)
> "컴프레서는 죽지 않는다 — 죽임당한다" (Compressors Don't Die - They're Killed)

### 🔴 컴프레서 5대 기계적 고장 원인 (기계 > 전기)

| # | 고장 유형 | 원인 | 증상 | 예방 |
|---|----------|------|------|------|
| 1 | **과열** (Overheating) | 높은 압축비, 오일 손실, 흡입라인 비단열 | DLT>225°F(107°C), 오일 윤활력 상실>300°F(149°C) | 정상 충전, 흡입라인 단열, 응축기 청소 |
| 2 | **플러드백** (Floodback) | 낮은 과열도, TXV 오버피딩, 에어플로우 부족 | 과열도≈0, 오일 세척, 베어링 마모 | 과열도 모니터링, 에어플로우 확인 |
| 3 | **리퀴드 슬러깅** (Slugging) | 냉매 마이그레이션→기동 시 폭발적 기화 | 금속 충격음, 피스톤/로드 파손, 3000psi+ | 크랭크케이스 히터, 펌프다운 사이클 |
| 4 | **플러디드 스타트** (Flooded Start) | 오프사이클 중 냉매→크랭크케이스 응축 | 기동 시 오일 폼, 베어링 워싱 | 크랭크케이스 히터(외기<12°C), 펌프다운 |
| 5 | **오일 로스** (Oil Loss) | 증발기 오일 체류, 짧은 사이클, 흡입라인 과대 | 윤활 부족, 과열, 소착 | 적정 충전, 오일 리턴 확인, 사이클 시간 |

### 📋 시스템 체크포인트 (Trevor Matthews 체크리스트)

**응축기 측:**
- [ ] 응축기 흡입공기 온도
- [ ] 응축기 배출공기 온도
- [ ] 리퀴드라인 압력 → 온도 변환
- [ ] 리퀴드라인 실측 온도
- [ ] 응축기 출구 과냉도

**필터/리퀴드라인:**
- [ ] 필터 입구 온도
- [ ] 필터 출구 온도 (ΔT>3°C면 막힘)
- [ ] 수분/기포 (사이트글라스)
- [ ] TXV 입구 과냉도

**증발기 측:**
- [ ] 증발기 흡입공기 온도
- [ ] 증발기 배출공기 온도
- [ ] 증발기 압력 → 포화온도 변환
- [ ] 증발기 출구 온도
- [ ] 증발기 과열도

**컴프레서:**
- [ ] 흡입압력 입/출
- [ ] 컴프레서 과열도 (~20°F/11K 목표)
- [ ] 토출온도 (<225°F/107°C 세미허메틱, <260°F/127°C 스크롤)
- [ ] 압축비 (AC 스크롤 최대 11:1, 냉동 스크롤 26:1)

## 9. HVAC School (Bryan Orr) — 현장 진단 5대 기둥

### 🏛️ 냉매 회로 진단 5대 기둥 (5 Pillars)

**모든 서비스 콜에서 이 5가지를 반드시 체크:**

| # | 측정값 | 알 수 있는 것 | 룰 오브 썸 |
|---|--------|-------------|-----------|
| 1 | **흡입압력 (Suction P)** | 증발기 내 냉매 끓는점 온도 | 실내온도 - 35°F = 포화온도 (400CFM/ton 기준) |
| 2 | **토출압력 (Head P)** | 응축기 상태, 압축비, 냉매 백업 | 실외온도 + 15~20°F = 포화온도 |
| 3 | **과냉도 (Subcooling)** | 리퀴드라인 상태 (전부 액체인지) | TXV 시스템: 8~14°F (4~8K) |
| 4 | **과열도 (Superheat)** | 증발기 피딩 상태, 컴프레서 보호 | TXV: 5~15°F (10±5), 고정오리피스: 차트 참조 |
| 5 | **델타T (ΔT)** | 시스템 전체 열교환 성능 | 공급~리턴 15~22°F (습도에 따라 변동) |

*+보너스: 정압(TESP) 0.3~0.7 inWC*

### ⚡ CTOA — 기기 효율별 응축온도 기준

| 효율 등급 | 연식 기준 | 응축온도 - 외기온도 (CTOA) |
|----------|----------|-------------------------|
| 6-10 SEER | ~1991 이전 | 30°F (17K) |
| 10-12 SEER | 1992~2005 | 25°F (14K) |
| 13-15 SEER | 2006~ | 20°F (11K) |
| 16+ SEER | 2006~ | 15°F (8K) |

→ **앱 기능**: 기기 연식/효율 입력 → 자동으로 정상 범위 계산

### 🔍 DTD — 증발기 설계온도차 (Evaporator Split)

- **400 CFM/ton**: DTD = 35°F (실내온도 - 흡입포화온도)
- **350 CFM/ton**: DTD = 38~40°F
- 과대 증발기 코일: DTD 약간 낮아짐 (흡입압력 높아짐)

### 🔬 NIST — Non-Invasive System Testing (비침습 시스템 테스트)

**혁명적 개념: 게이지 연결 없이 시스템 진단**

**왜 중요한가:**
- 게이지 연결 1회당 냉매 5% 손실 (주거용)
- 전 세계 냉매 방출의 50%가 "게이지 연결"에서 발생
- 전 세계 20억대 에어컨, 총 냉매 뱅크 = 자동차 50억대 연간 배출과 동일

**3가지 NIST 방법론:**

| 방법 | 개발/기관 | 특징 |
|------|----------|------|
| ANSI/ACCA 310 | 미국 국가 표준 | 신규 설치용, CFM + 온도 측정 |
| Mowris NTD | Robert Mowris 특허 (2023) | DTD + 온도 관계 + 전력 소비 |
| measureQuick Benchmarking | Jim Bergmann | 클라우드 기반 "알려진 정상" 스냅샷 저장 |

### 📐 NIST 실전 워크스루 (3톤 R-410A TXV 시스템)

**조건: 외기 85°F**

**Step 1: 에어플로우 확인**
- 목표: 1200 CFM (400 × 3톤)
- 실측: 1150 CFM → ±5% 이내 → OK

**Step 2: 설계 온도차 확인**
- 증발기 DTD = 35°F (13-14 SEER)
- 응축기 CTOA = 20°F (13-14 SEER), 고효율은 15°F

**Step 3: 온도 측정**
- 리턴 건구: 75°F / 습구: 63°F
- 공급: 55°F
- 리퀴드라인: 95°F
- 석션라인: 53°F

**Step 4: 수학 (게이지 없이!)**

증발기 쪽:
```
예상 코일온도 = 75 - 35 = 40°F
+ 목표 과열도 10°F = 50°F 예상 석션라인
실측 53°F → 차이 3°F → ✅ 정상
```

응축기 쪽:
```
예상 응축온도 = 85 + 20 = 105°F
- 목표 과냉도 10°F = 95°F 예상 리퀴드라인
실측 95°F → 정확 일치 → ✅ 정상
```

**Step 5: ΔT 확인**
```
75 - 55 = 20°F → 습구 63°F 기준 정상
```

**→ 앱 핵심 기능: "게이지 없이 진단" 모드!**
- 온도만 입력 → DTD/CTOA 자동 계산 → 정상/이상 판정
- 경쟁 앱 중 이 기능 있는 곳 = 0

### 🛡️ ABC 원칙 — Airflow Before Charge

"게이지 잡기 전에 에어플로우부터!"
- 95%의 경우 게이지 연결 불필요

- 에어플로우 문제 해결 후에도 안 되면 그때 게이지

# 🔧 소스 2: Sporlan/Parker 공식 TXV 트러블슈팅 (Bulletin 10-143)

## "12 Solutions for Fixing Common TEV Problems"

### TXV 기본 원리 복습 (Sporlan 공식)
- 다이어프램에 3가지 압력 작용:
  - **P1 (센싱벌브 압력)** → 석션라인 온도 비례 → 개방력
  - **P2 (이퀄라이저 압력)** → 증발기 압력 → 폐쇄력
  - **P3 (스프링 압력)** → 과열도 설정값 → 폐쇄력
- 정상 상태: P1 = P2 + P3
- **스프링 압력은 일단 설정되면 상수** → TXV는 실질적으로 벌브 압력과 이퀄라이저 압력의 차이(= 과열도)를 제어

### 밸브가 냉매를 충분히 안 먹일 때 (언더피딩) — 7가지 체크

#### #1. TEV 조정 확인
- 공장 세팅 = 중간 위치 (center stem)
- 전좌 → 후좌 총 회전수 세고, 50% 지점으로 설정
- **반시계 방향**으로 1/2~1회전씩, **15분 간격**으로 조정
- 올바른 과열도 도달까지

#### #2. 센싱 벌브 위치 확인
- 깨끗하고 직선인 석션라인 수평부에 단단히 고정
- 증발기 출구의 수평 석션라인에 장착
- **7/8" 이상 라인**: 수평관 **4시 또는 8시 방향**에 설치
- 작은 라인: 어디든 가능하지만 **바닥(6시)은 절대 금지** (오일 영향)
- 멀티 증발기: 한 밸브의 흐름이 다른 밸브 벌브에 영향 주지 않게 배관
- 이퀄라이저 연결: 벌브에서 수 인치 하류

#### #3. 수분(Moisture) 확인
- 물 또는 물+오일 혼합물이 밸브 포트에 얼어 작동 방해
- TEV가 시스템에서 **첫 번째 저온 지점** → 수분이 여기서 먼저 얼음
- 시스템을 수년간 안 열었어도 수분 문제 가능!
  - 리퀴드라인 드라이어가 수분 보유 한계 도달 후 온도 상승 → 수분 방출
  - 냉매 누설, 더러운 응축기, 응축기팬 고장 → 리퀴드라인 온도 상승
- **진단법**: 밸브 본체에 뜨거운 물 한 컵 부으면 → 냉매 흐름 소리 급증 = 수분 확인
- **조치**: 새 Catch-All 필터-드라이어 설치

#### #4. 밸브 본체 내 오염물 확인
- 구리 산화물, 금속 조각, 오일 분해물, 슬러지 등
- 일반 스트레이너는 이런 미세 오염물을 통과시킴
- **조치**: 펌프다운 → TXV 분해 청소 → 필터-드라이어 설치
- 오염물 없으면 → 사이트글라스 설치 후 #5로

#### #5. TEV 입구에 기포 없는 순수 액체 확인
- 리퀴드라인에 플래시가스 있으면 TXV 용량 저하
- **원인**: 배관 크기 부적절, 수직 리프트, 과도한 압력 강하
- **조치**: 열교환기 설치로 과냉도 확보, 배관 교정

#### #6. 설계 압력강하 대비 TEV 용량 확인
- TEV 용량 = 밸브 전후 압력차에 따른 변수
- 입구 압력 손실 또는 출구 압력 상승 제거
- 저응축 압력 → 헤드 프레셔 컨트롤 설치
- **더 큰 TEV 설치는 최후의 수단**

#### #7. 엘리먼트 차지 마이그레이션 확인
- 가스형 차지(ZP, CP, VGA)는 제한된 양의 구성물
- **벌브 온도 < 엘리먼트 온도**여야 정상 작동
- 반대가 되면 구성물이 벌브 → 엘리먼트로 이동 → 밸브 닫힘
- **조치**: 열풍기(로즈버드 토치 금지!)로 엘리먼트 가열 → 밸브 재위치

### 밸브가 증발기를 플러딩할 때 (오버피딩) — 5가지 체크

#### #8. TEV 조정 확인 (과냉 방향)
- **시계 방향**으로 1/2~1회전씩, 15분 간격 조정
- 올바른 과열도 도달까지

#### #9. TEV 입구 액체 상태 변화 확인
- 플래시가스 보상으로 밸브를 과도 개방 → 응축온도 하락 시 플러딩
- 부하 감소 시 순수 액체 공급 → 기존에 없던 플러딩 발생 가능
- **조치**: 사이트글라스 확인 + 조정

#### #10. 플러딩 사이클 타임 스터디
- 심한 헌팅: 과열도 0°F ↔ 10°F+ 반복 (수분 주기)
- **역직관적 해결**: 조정 스프링을 **느슨하게** (개방 방향)
  - 더 많은 유량 → 더 안정적인 피딩 → 헌팅 감소
  - 높은 과열도와 0°F 바닥 모두 개선

#### #11. 증발기 에어사이드 열부하 확인
- Walk-in cooler 공기순환 횟수, 냉동 진열대 풍속 감소 → 열전달 손실
- 먼지, 얼음, 빠진 판금 패널, 잘못된 제품 적재 → TXV 헌팅/플러딩
- **조치**: 에어 무브먼트가 설계 조건인지 확인 (ABC 원칙!)

#### #12. 불균등 증발기 회로 부하 확인
- 멀티회로 증발기에서 각 회로 열부하가 달라지면
  - 저부하 회로 → 리퀴드 스필오버 → 석션 온도 ↓ → TXV 쓰로틀
  - 정상 부하 회로 → 냉매 부족 → 냉장면적 손실
- **진단법**: 각 디스트리뷰터 회로 석션 출구 온도 비교 → 온도 불균등 = 불균등 부하
- **조치**: 정확한 디스트리뷰터/노즐 어셈블리 확인

### Trane 칠러 특화 진단 (RTAA/CGAM/CGAF)

#### Low Superheat 진단 (Trane RTAA)
- 과열도 ≤ 2°F가 2400 degree·F·seconds 이상 지속 → Low Superheat 경보
- RTAA 70-125: 과열도 설정값 4°F
- RTAA 130-400: 과열도 설정값 8°F
- **체크리스트:**
  1. 포화증발기 온도센서 + 컴프레서 석션센서 정확도 확인 (옴 체크, 접지 확인)
  2. EXV 정상 작동 확인: 과열도가 설정값 ±2°F 이내 유지되는지
  3. 과열도 매우 불규칙 + ±2°F 이탈 → EXV 불량 의심
  4. EXV 추가 테스트: 칠러 운전 후 정지 → EXV 폐쇄 명령 → 사이트글라스에서 냉매 흐름 없어야 정상
  5. 냉매 흐름 지속 → EXV 교체

#### Limp Mode (CGAM 등)
- 높거나 낮은 과열도가 20분 이상 감지 → 컴프레서 속도 제한
- 센서 고장으로도 트리거 가능
- 확인사항: Dome 센서, Suction 센서, Ambient 센서, Indoor 센서

#### Suction Pressure Sensor (CGAF)
- 0 PSIG = 0.118 VDC
- 100 PSIG = 4.847 VDC
- Superheat/Subcooling 옵션 설치 시에만 존재

### Carrier 히트펌프 진단 규칙 (난방 모드)

#### 난방모드 충전 확인 기본 원칙
- TXV 시스템: **과냉도로 충전** (냉방 10°F / 난방 14°F)
- 오리피스 시스템: **과열도로 충전** (목표 15°F)
- **반드시 적정 에어플로우 확인 후!**

#### 히트펌프 난방모드 경험식
| 항목 | 경험식 | 비고 |
|------|--------|------|
| 토출라인 온도 | 외기온도 + 100~110°F | 과열 = 부족, 저온 = 과충전 |
| 석션포화온도 | 외기온도 - 20~25°F | R-410A 기준 |
| 응축온도 (실내) | 실내온도 + 30~40°F | SEER/코일 크기 따라 변동 |

#### 히트펌프 충전 절차 (겨울)
1. 서리 완전 제거 (디프로스트 완료 확인)
2. 필터, 코일, 블로워 휠 청결 확인
3. 제조사 사양 확인 (Lennox: 65°F 이하 시 특수 절차)
4. 응축기 코일 일부 가림 → 냉방 모드로 충전 (일부 제조사)
5. 과열도/과냉도 안정화 후 최종 확인

## 핵심 진단 철학

### "Big Picture Diagnosis" — 증상 치료가 아닌 문제 해결
> "항상 큰 그림 진단에 집중하라 — 증상만 치료하지 말고 문제 자체를 해결하라"

### 5가지 핵심 진단 원칙 (현장 실전)

#### 1. 압력만으로는 충분하지 않다
- 흡입압력이 낮다고 바로 냉매를 추가하지 마라
- 낮은 흡입압의 3대 원인:
  - 냉매 부족 (누설)
  - 에어플로우 부족 (필터/코일)
  - 계량장치 제한 (TXV/드라이어)
- **과열도 + 과냉도를 함께 봐야** 정확한 진단

#### 2. 과열도와 과냉도는 냉매 위치 지도
- **과열도** = 증발기 내 냉매가 충분한지 (높음 = 부족, 낮음 = 넘침)
- **과냉도** = 응축기 내 냉매가 충분한지 (높음 = 과다, 낮음 = 부족)
- 조합으로 읽어야 한다:

| 과열도 | 과냉도 | 의미 |
|--------|--------|------|
| 높음 | 낮음 | 냉매가 응축기에도 부족 → 누설/부족 충전 |
| 높음 | 높음 | 냉매가 응축기에 갇힘 → 계량장치 제한 |
| 낮음 | 높음 | 냉매가 너무 많음 → 과충전 |
| 낮음 | 낮음 | 컴프레서가 못 뽑음 → 컴프레서 불량 |

#### 3. "Gas-n-Go"는 절대 하지 마라
- 냉매 추가 후 도망 → 90% 확률로 1주 내 재방문
- 반드시 **누설을 먼저 찾아 수리** → 그 다음 충전
- "처음부터 제대로 하는 게 훨씬 쉽다"

#### 4. 필터 드라이어를 의심하라
- 특히 컴프레서 교체 이력이 있는 시스템
- 필터 드라이어 전후 온도차 > 3°F = 제한
- 그러나 시스템 운전 초기에 측정해야 (나중에는 전체 라인이 차가워짐)

#### 5. 데이터를 기록하고 분석하라
- 현장에서 측정값을 꼼꼼히 기록
- "과거 데이터가 있으면 진단 시간이 절반으로 줄어든다"
- → **앱의 '서비스 이력 기록' 기능 가치 입증**

## 실전 케이스 스터디 (포럼/현장 종합)

### 케이스 1: 40톤 McQuay 칠러 — 2년간 컴프레서 8개 교체

**상황:**
- Freezestat 경보로 정지
- 리셋 후 게이지 연결: 80/250 (정상적으로 보임)
- 양쪽 컴프레서 석션라인에 심한 결로 (sweating)

**일반 기사 판단:** "석션라인 잘 식고 있으니 괜찮아"

**실제 측정:**
- 과열도: 35°F (정상 범위 5~15°F의 3배!)
- 과냉도: 55°F (정상 범위 8~14°F의 4배!)
- 외기 습구온도: 70°F

**진단:** 고과열도 + 고과냉도 = **계량장치(TXV) 심각한 제한**
- 냉매가 응축기에 갇혀서 고과냉도
- 증발기에 냉매 못 들어가서 고과열도
- 컴프레서가 과열 → 반복 파손의 근본 원인

**해결:** TXV 교체/수리 → 컴프레서 파손 반복 해결

**앱 레슨:** 압력만 보면 "정상"으로 보이는데, 과열도/과냉도가 진실을 말한다

### 케이스 2: Trane 유닛 — 증발기 결빙 + 이상 압력

**상황:**
- 도착 시 증발기 완전 결빙
- 해빙 후 필터 교체, 코일 상태 확인 (비교적 깨끗)
- 압력: 흡입 45 psi / 토출 185 psi (90°F+ 외기)

**1차 기사 판단:** 냉매 부족 → 충전 추가
- 토출압 285까지 올림
- 흡입압 50 이상 안 올라감
- 과열도 18°F 이하로 안 내려감

**포럼 전문가 진단:**
- 고과냉도 의심 → **필터 드라이어 제한** (Trane 공장 구리 드라이어)
- 또는 오리피스/피스톤 막힘
- "40°F 과냉도 + 사이트글라스 플래싱 = 리퀴드라인에 심각한 제한"
- 이미 과충전 상태가 되었을 가능성

**올바른 조치:**
1. 냉매 회수
2. 필터 드라이어 교체
3. 새 냉매 무게 충전
4. TXV 시스템이면 과냉도 10~15°F로 충전

**앱 레슨:** 
- 냉매 추가 전 반드시 과냉도 확인
- 필터 드라이어 ΔT 체크는 필수
- "Gas-n-Go"의 위험성 — 과충전으로 악화

### 케이스 3: 히트펌프 겨울 서비스콜

**상황:**
- R-410A 히트펌프, 외기 40°F
- 석션라인 뜨거움, 흡입압 20 psi, 토출압 82 psi

**진단:**
- 20 psig R-410A = 포화온도 약 -2°F (매우 낮음)
- 82 psig 토출 = 포화온도 약 35°F (매우 낮음)
- 100~110°F over ambient 규칙: 40 + 100 = 140°F 예상인데 토출라인 온도 낮음

**포럼 전문가 의견:**
- 심각한 냉매 부족 (undercharge)
- 과냉도 측정 필수 (리퀴드라인 온도 + 압력)
- "먼저 누설 찾아 수리, 회수 후 무게 충전"
